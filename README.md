# Random-password-generator

Description

This is a simple Python program that generates a random password based on the length entered by the user.

The program uses:

- Letters (A-Z, a-z)
- Numbers (0-9)
- Special characters (!, @, #, $, etc.)

to create a strong random password.

Features

- User chooses the password length.
- Generates a random password.
- Includes letters, digits, and symbols.
- Easy to understand for Python beginners.

Requirements

- Python 3.x

How to Run

1. Open a terminal or command prompt.
2. Navigate to the project folder.
3. Run the program:

python password_generator.py

4. Enter the desired password length.
5. The program will display a randomly generated password.

Example

Enter digit: 12

Generated Password:
aB7@kP!2x#Q$

Technologies Used

- Python
- random module
- string module
